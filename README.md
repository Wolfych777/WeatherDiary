## Описание программы
Weather Diary — это GUI-приложение для ведения дневника погоды. Приложение написано на Python с использованием библиотеки Tkinter.

### Возможности:
Программа позволяет:
- добавлять записи о погоде;
- просматривать записи в таблице;
- фильтровать записи:
 - по дате;
 - по температуре;
- сохранять данные в JSON;
- загружать данные из JSON;
- проверять корректность введённых данных.

## Примеры использования

### Добавление записи
1. Заполните все поля формы:
    **Дата**	2026-05-12;
    **Температура**	10;
    **Описание**	Солнечно;
    **Осадки**	Да/Нет.

2. Нажать:
    *Добавить запись*

3. Результат:
    запись появится в таблице.

### Фильтрация записей:
**Фильтр по дате** Позволяет показать записи только за выбранный день.
Пример:
   В поле *Фильтр по дате* вводим нужную дату: "2026-05-12", после нажимаем кнопку *Применить*, в таблице останутся только записи за эту дату.
   
**Фильтр по темепературе** Позволяет показать записи с температурой выше указанного значения. 

Пример:
    В поле *Температура выше* вводим нужные параметры температуры: "10", после нажимаем кнопку *Применить*, после применения фильтра будут показаны записи c температурой выше 10: 
11°C;
15°C;
22°C. Записи с температурой ниже или равной 10°C отображаться не будут.

## JSON 

### Сохранение JSON
Кнопка: *Сохранить JSON*
сохраняет все записи в файл формата .json.

### Загрузка JSON
Кнопка: *Загрузить JSON*
загружает ранее сохранённые данные обратно в программу.

### Требования
Для работы программы необходимо:
Python 3.10 или выше

1. Скачать проект:
    скачать ZIP-архив с GitHub;

3. Перейти в папку проекта:

**Windows** cd C:\Users\User\Desktop\weather-diary.
**Linux / macOS** cd ~/Desktop/weather-diary.
Запустить файл:
```bash
python main.py
```
